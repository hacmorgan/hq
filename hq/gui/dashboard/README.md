# Dashboard

# Running

## Running the server
The server is executable, simply run the file:
```bash
hq/gui/dashboard/lib/server.py
```

## Running the client
The client is run using flutter from the project root
```bash
(cd hq/gui/dashboard && flutter run -d chrome)
